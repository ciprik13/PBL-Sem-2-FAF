Below is a model of the map of Vienna airport, represented as a graph, where Dijkstra’s algorithm was applied to calculate the shortest path for various operations. The airport terminal is mapped out in the graph where nodes symbolize critical locations within the airport’s system. These key locations include passenger boarding gates, baggage handling areas, maintenance zones, ground transportation stops, and other pivotal points that facilitate the smooth transition of passengers, luggage, and equipment through the airport. The edges between these nodes are carefully designed to represent paths or connections, ensuring a clear and efficient layout for movement and operation

Dijkstra’s Algorithm applied to find the shortest path
